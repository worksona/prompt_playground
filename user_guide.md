## 2. Managing Content

### Upload Content
1. In the left sidebar, find "üìö Upload Content"
2. Click "üìé Upload File or Image" to:
   - Upload text files (.txt, .md)

2. For each document you can:
   - Preview content
   - Use content as source

### Persona Cards
1. Click "üë• Select Audience..." dropdown
2. Choose a persona template
3. Click "+" to add the persona card
4. Customize:
   - Name
   - Description
   - Direction (From/To)

### Prompt Cards
1. Click "‚ú® Select Prompt..." dropdown
2. Choose a prompt template
3. Click "+" to add the prompt card
4. Configure the prompt:

5. Click "Run" or "Analyze Image" to generate results
6. Use the result actions:
   - Enhance content
   - Copy to clipboard
   - Clear result
